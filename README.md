<h3>Setup</h3>
Get an API key from TheMovieDatabase at:<br>
https://www.themoviedb.org/login
<br>
Run <code>bundle install</code>
<br>
Run <code>rails webpacker:install</code>

<h3>Run</h3>

Run <code>rails server</code>
<br>
Open <code>http://localhost:3000</code>
<br>
